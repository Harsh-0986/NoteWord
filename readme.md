# Google_Doc_Clone
A fully functional Google Doc Clone

To use the clone
Install PyQt5 and docx2txt 

Commands:

For Windows:
```
pip install pyqt5 docx2txt
```

For Linux:
```
pip3 install pyqt5 docx2txt
```

To Run the script:

For Windows:
```
python main.py
```

For Linux:
```
python3 main.py
```
